$ omakase

Post-installation scripts for Linux distributions

Introduction

Just like DHH's Omacom philosophy, I've chosen to collect all my post-installation scripts for Linux distributions in one place.
This is Omakase - a curated collection of opinionated scripts that automate the setup of popular Linux distributions. With a single command, you can save time and get a ready-to-use system tailored to my preferences and workflow.

$ ls omakase/

> omarell

Basically Omakub with some fancy, totally optional (... and unrequested!) tweaks

wget -qO- https://omakase.lucapattocchio.dev/omarell | bash

> omasway

Opinionated Ubuntu/Sway Setup

wget -qO- https://omakase.lucapattocchio.dev/omasway | bash
